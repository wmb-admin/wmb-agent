# FrontendUISkill

- 中文名称: 前端UI样式增强
- 所属分组: （二）前端相关Skill
- 来源Prompt: 美业SaaS自动迭代智能体完整版Prompt
- 模板说明: 以下输入、输出、约束与交接项基于原始 Prompt 结构化整理，便于接入 Agent。

## 核心目标

在不破坏业务逻辑的前提下提升页面到商用级视觉质量。

## 触发时机

- 页面功能开发完成后
- 需要统一全站视觉风格、间距、层级和状态样式时

## 输入

- 已完成功能开发的页面与组件
- 现有设计风格和品牌约束
- 需要增强的按钮、表单、表格、卡片、空态与加载态

## 输出

- 优化后的页面样式实现
- 统一的视觉风格说明
- 供 FrontendTestSkill 验证的样式增强结果

## 硬性约束

- 只做样式增强，不改变页面业务逻辑
- 不破坏既有布局结构和交互流程
- 需兼顾页面间距、对齐、留白、层级、阴影和过渡表现

## 执行清单

1. 统一页面间距、栅格、留白和层级
2. 优化按钮、表单、表格、卡片和状态样式
3. 补足加载态、空态和过渡动效
4. 确认增强结果不影响业务行为

## 交接对象

- FrontendTestSkill

## 原始 Prompt

对页面做商用级美观优化，保证全站视觉风格统一
优化页面间距、对齐、留白、层级、阴影、过渡动画
美化按钮、表单、表格、卡片、加载/空状态样式
仅做样式增强，不破坏原有页面布局与业务逻辑
