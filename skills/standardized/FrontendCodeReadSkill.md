# FrontendCodeReadSkill

- 中文名称: 前端代码阅读
- 所属分组: （二）前端相关Skill
- 来源Prompt: 美业SaaS自动迭代智能体完整版Prompt
- 模板说明: 以下输入、输出、约束与交接项基于原始 Prompt 结构化整理，便于接入 Agent。

## 核心目标

只读取前端代码并理解页面、组件、接口封装、路由和权限体系。

## 触发时机

- 开始新增或改造前端页面前
- 需要确认样式规范、组件复用和前端版本信息时

## 输入

- 当前迭代版本号
- 目标页面或模块范围
- Vue3 页面、组件、API 封装和路由目录
- 现有权限和全局配置

## 输出

- 前端页面结构与路由摘要
- 接口调用方式与权限展示规则
- 组件复用点与样式规范说明
- 供 FrontendCodeWriteSkill 使用的上下文结论

## 硬性约束

- 只读取前端代码，不读取后端 Java 文件
- 不得臆造不存在的路由、组件或接口字段
- 必须确认前端版本展示要求和风格约束

## 执行清单

1. 定位页面、组件、API、路由、权限和全局配置文件
2. 识别页面布局、表单表格模式和组件复用规则
3. 梳理当前前端版本和样式规范
4. 整理后续开发需要的结构化上下文

## 交接对象

- FrontendCodeWriteSkill
- FrontendUISkill

## 原始 Prompt

读取Vue3页面、组件、API封装、路由、权限、全局配置文件
识别前端版本号、项目样式规范、组件复用规则
绝不读取任何后端Java代码
